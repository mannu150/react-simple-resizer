# Simple Resizer

A library for resizing a child element either vertically or horizontally or both based on props provided.


## To run the project
```
git clone https://github.com/mannu150/react-simple-resizer.git
npm install
npm start
```

## To deploy docs
```
npm run deploy:docs
```

## To run unit tests 
```
npm t
```
